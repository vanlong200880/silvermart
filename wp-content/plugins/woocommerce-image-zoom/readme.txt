WooCommerce Image Zoom add zoom feature to product images with LightBox popup image gallery.

== Description ==

Running an eCommerce site with WooCommerce? The WooCommerce product image zoom is most necessary for your site. The more you can show your product details to your customers more you can get conversions. Woocommerce Image Zoom allows you to magnify product images and set zooming feature easily and quickly.

It's premium version Woocommerce Image Zoom PRO comes with an advance setting panel. So you can configure the magnifier as you need.

### Plugin Features

* Show a bigger size product image on mouseover.
* Plug & Play.
* No need any extra settings.
* Very light weight.
* Work with all Wordpress theme.
* Easy to use.
* Developer friendly & easy to customize.
* Powered by Elevate Zoom.

* Support product variation images zoom.

== Installation ==

* Install it as a regular WordPress plugin

* Active the plugin.

* Go to any single product page of Woocommerce product.

* Hover product image.



== Frequently asked questions ==

= How can I setup zoom to product image? =
You don't have to do any setting, just active this plugin and check your single product page image.
